iPhone / iPad

Safari → www.cognethra.org/app/ → Κοινοποίηση → Προσθήκη στην Αρχική Οθόνη

Σε αυτό το iPhone: πατήστε Κοινοποίηση στο Safari, μετά Προσθήκη στην Αρχική Οθόνη.

Android

Chrome → www.cognethra.org/app/ → μενού → Εγκατάσταση εφαρμογής ή Προσθήκη στην αρχική οθόνη

Chrome ή Edge → εικονίδιο εγκατάστασης στη γραμμή διευθύνσεων, ή σελιδοδείκτης www.cognethra.org/app/

Η καταχώριση στο App Store προγραμματίζεται για μεταγενέστερη φάση. Η Προσθήκη στην Αρχική Οθόνη λειτουργεί σήμερα.
