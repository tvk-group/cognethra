インストール方法

iPhone / iPad

Safari → www.cognethra.org/app/ → 共有 → ホーム画面に追加

iPhone / iPad

この iPhone では：Safari で共有をタップし、ホーム画面に追加。

Android

Chrome → www.cognethra.org/app/ → メニュー → アプリをインストールまたはホーム画面に追加

デスクトップ

Chrome または Edge → アドレスバーのインストールアイコン、または www.cognethra.org/app/ をブックマーク

App Store 掲載は後期フェーズで計画。ホーム画面に追加は現在利用可能。
